== Frequently Asked Questions ==
= What is loginMojo? =
loginMojo is an innovative product of Version# Technologies LLP. It can be integrated with any client facing website or application to allow one more login option as – ‘Login with WhatsApp’ to your clients using official WhatsApp Business API.

Using this option, your customers can login into your website / apps in just 2 clicks using ‘Login with WhatsApp’ without entering anything.

The only condition is your clients WhatsApp number should be registered in your records.

This feature is widely used as an alternate to SMS / OTP verification.

= Do you provide ‘Login with WhatsApp’ feature ? =
Yes, you are at right place if you are looking for the solution to allow your customers to ‘Login with WhatsApp’.
At loginMojo, we provide such an unique solution which can be easily integrated in your client facing websites / apps.
Check our Technical support section for detailed technical document for further details.  https://loginmojo.com/faqs

= Can we create multiple API key from loginMojo? =
Yes.

= Is this official integration with WhatsApp Business API ? =
Yes, we have officially tie-up to use WhatsApp Business API for such kind of integrations. We understand how important our customers are for us and this entire solution has been designed keeping a long lasting relationship in mind.

= How to generate loginMojo API key ? =
You need to follow below easy steps to generate loginMojo API key :
Go to API Token page.
Click on ‘Create’ button to generate API Token.
Sample Token : CS2MYVkbdea2cbFM4tGLubEvpG5BbSOe7Nxsh4sy 
Copy the generated token and store it in a safe way, for your security, it won’t be shown again.

= Can we pass extra parameters during token request? =
Yes, you may pass your parameters in the meta array during token request and that meta array we will send during the webhook call.
